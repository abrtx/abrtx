# Designing software that embraces change.

## About Me

I'm a software engineer passionate about transforming ideas into systems that continue to evolve.

I believe software should be designed with change in mind from the very beginning. My work focuses on defining clear architectures before implementation, creating maintainable systems through well-defined responsibilities, modular design, and thoughtful engineering decisions.

I enjoy working across backend systems, embedded software, and robotics because each discipline provides a different perspective on building reliable, maintainable, and extensible software.

---

## Engineering Philosophy

> **Engineering transforms ideas into reality.**
> **Architecture gives those ideas the ability to evolve.**

To me, software is never truly finished. Every system should remain understandable, maintainable, and ready for future change.

I believe maintainability is not a feature added at the end of a project—it is a responsibility considered from the very first design decision.
